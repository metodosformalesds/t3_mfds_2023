¿Quienes somos?

TicketJaeger es una nueva opcion para todo aquel que busque boletos para sus eventos favoritos, esta diseñado para ser facil de entender y navegar, esperamos que TicketJaeger te sea util para crear y organizar tus eventos; o que te sirva para encontrar los eventos que buscas de manera sencilla. Jaeger significa cazador en aleman, y asi como un cazador puedes usar TicketJaeger para cazar los evento que quieres.


Preguntas frecuentes

¿Cuesta hacer una cuenta en TicketJaeger?
-No, registrarse como usuario es completamente gratis

¿Cual es la diferencia entre una cuenta de comprador y una de organizador?
-Segun el tipo de usuario se pueden hacer distintas acciones, los compradores pueden buscar entre los eventos disponilbles para escoger boletos, agregarlos a su carrito y comprarlos

¿Podemos esperar mas de TicketJaeger?
-Si, el sistema esta en sus primeros dias de desarrollo y estamos trabajando para traer mayor funcionalidad de la manera mas eficiente posible
